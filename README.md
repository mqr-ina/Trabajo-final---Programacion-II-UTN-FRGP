# 🎬 Sistema de Gestión de Salas de Cine

Trabajo Final - Programación II | UTN FRGP

## 📋 Descripción
Este sistema permite administrar salas de cine, facilitando el control
de películas, salas y entradas. Desarrollado en C++ aplicando conceptos
de programación orientada a objetos.

## 🛠️ Conceptos aplicados
- Clases y Objetos
- ABM (Alta, Baja y Modificación)
- Lógica de programación
- Archivos y persistencia de datos

## ⚙️ Requisitos
- [Code::Blocks](https://www.codeblocks.org/downloads/) con MinGW

## 🚀 Cómo ejecutar
1. Clonar el repositorio
2. Abrir el archivo `.cbp` con Code::Blocks
3. Compilar y ejecutar con **F9**

## 📁 Estructura del proyecto
- `GestorPelicula` - Gestión de películas
- `GestorSala` - Gestión de salas
- `GestorArchivos` - Manejo de archivos
- `Menu` - Interfaz de usuario
